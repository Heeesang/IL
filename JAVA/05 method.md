# Method

- 입력을 받고 어떤 일을 수행한 다음에 결과물을 내놓는것<br>
- 특정 기능을 정의한 코드들의 집합

## 메소드의 정의

```java
/* public static 리턴타입 메소드명(){
    실행할 코드
    return ;
}*/

public static void Hello(){
    System.out.println("Hello!");
}

 public static void main(String[] args){
//메서드 호출
    Hello();
 }
```

## 입력과 출력

- 매개변수: 외부로부터 입력값(인자)을 받기위한 변수
- 인자: 어떤 함수를 호출시에 전달되는 값

## 메소드를 사용하는 이유

- 코드 중복 방지
- 가독성 up!
- 손쉬운 유지 보수
