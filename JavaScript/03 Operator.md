# 연산자

> 특정 연산을 할때 사용하는 문자

```jsx
let value = 1; // '=' 대입 연산자
let a = 1 + 2; // 3
let b = 1 + 2 - (3 * 1) / 3;
```

-증감 연산자

```jsx
let a = 1;
console.log(a++); // 전위 1
console.log(a); // 2
console.log(++a); // 후위 3
```

-대입 연산자

```jsx
let a = 1;
a *= 3;
console.log(a); // 3
a += 3;
console.log(a); // 6
```
